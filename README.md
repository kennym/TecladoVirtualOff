TecladoVirtualOff
=================

¿Cuántas veces has pensado que el teclado virtual es más
rompebolas que útil?

Esta extensión para Chrome desactiva el teclado virtual en los
sitios de los siguientes bancos del Paraguay:

  - Banco Visión
  - Banco Familiar
  - Itau
